From Prompt to Response

Every time you send a message to an LLM, it goes through these six stages in milliseconds.

  1. 1

    User Input

    The user types a prompt into the interface. This raw text is the starting point for everything that follows.

  2. 2

    Tokenisation

    The text is split into tokens — small chunks of roughly 3–4 characters each. The model never sees raw words, only these numeric token IDs.

  3. 3

    Embedding

    Each token ID is converted into a high-dimensional vector — a list of numbers that encodes the token's meaning and position in the sequence.

  4. 4

    Attention Layers

    The vectors pass through many Transformer layers. Each layer uses an "attention" mechanism to let every token look at every other token and update its representation based on context.

  5. 5

    Prediction

    After the final layer, the model produces a probability distribution over its entire vocabulary — essentially a ranked list of every possible next token.

  6. 6

    Output Generation

    The model samples from that distribution to pick the next token, appends it to the sequence, and repeats the whole process until it generates an end-of-sequence token or hits the output limit.
